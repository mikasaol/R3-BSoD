# R3-BSoD

## Five ways to create a BSoD in R3 (Windows)

1. NtRaiseHardError
2. RtlSetProcessIsCritical
3. NtSetInformationProcess
4. RtlSetThreadIsCritical
5. NtSetInformationThread

## Impact version

Windows XP-Win11(x86 and x64)
